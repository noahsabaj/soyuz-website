# Soyuz Website

Documentation and landing page for the Soyuz project.

## Tech Stack

- SvelteKit
- Tailwind CSS v4
- npm

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Notes

- API docs import `SOYUZ_API.json` directly from the parent `soyuz/` directory
- Requires the `soyuz` repo to be present at `../soyuz` for docs imports to work
